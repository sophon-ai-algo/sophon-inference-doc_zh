SAIL Python API
===============

SAIL use "pybind11" to wrap python interfaces, support python3.5.

Basic function
_____

.. code-block:: python

        def get_available_tpu_num():
            """ Get the number of available TPUs.

            Returns
            -------
            tpu_num : int
                Number of available TPUs
            """
